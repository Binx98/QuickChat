### QuickChat介绍

Web实时聊天IM通讯系统，通过Netty基于WebSocket协议构建通讯服务器，功能丰富好玩

1. 支持单聊、群聊，孩子乐意咋聊咋聊
2. 支持语音通话、视频通话，面对面实时畅聊
3. 支持PC、移动多终端适配，满足各种使用场景
4. 支持图片、文件、视频、文字、表情包等多种消息媒体
5. 白天、夜间模式自由切换
