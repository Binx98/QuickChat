## ✨系统介绍

功能丰富的实时IM通讯系统，通过Netty构建高性能WebSocket服务器。<br>
具有单聊、群聊、文件传输、语音消息、面对面音频视频通话等强大功能。<br>
支持任意联网设备直接通过浏览器随时随地访问，消息多端系统实时同步。<br>
