### QuickChat介绍

Web实时聊天IM通讯系统，通过Netty基于WebSocket协议构建通讯服务器，功能非常丰富

1. 单聊、群聊，孩子乐意咋聊咋聊
2. 语音、视频通话，面对面畅聊
3. PC、移动都适配，随时随地自由切换
4. 图片、文件面对面快速传输
5. 白天、夜间模式自由切换
