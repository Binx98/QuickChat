#### QuickChat介绍
Web实时聊天IM通讯系统，通过Netty基于WebSocket协议构建通讯服务器，功能非常丰富
1. 单聊群聊，孩子乐意咋聊咋聊
2. 视频通话，1v1尽情畅聊
3. PC端移动端自适应，多种终端都可适配
4. 图片、文件面对面传输
5. 多种多样聊天信息
6. 白天、夜间模式自由切换，
7. 自定义diy主题，个性化
